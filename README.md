## Requisitos
- Node.js ou Bun
- npm, yarn, pnpm ou bun

## Como Iniciar
- Instale as dependências:

```
npm install
# ou
yarn install
# ou
pnpm install
# ou
bun install

```
- Inicie o servidor de desenvolvimento:

```
npm run dev
# ou
yarn dev
# ou
pnpm dev
# ou
bun dev

```
## Stack Utilizada
- Framework: React
- Estilização: Tailwind CSS
- Backend: Firebase
